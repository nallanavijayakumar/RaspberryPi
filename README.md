# RaspberryPi

Use following command to download and extract the repository

    wget -q -O master.zip https://github.com/nallanavijayakumar/RaspberryPi/archive/master.zip  && \
    unzip master.zip && \
    rm master.zip -f && \
    cd ./RaspberryPi-master/installer_scripts && \
    sudo ./installer.sh
